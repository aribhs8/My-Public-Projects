Introducing FAVOC

FAVOC is a revolutionary new product that simplifies the home leisure experience.

Ultra Portable

Personal Connection

Features

Different from the competition


Image Processing

FAVOC uses real-time image processing to identify your position and your movement. Allowing the fan to follow you no matter what activity you are doing.


Portability Redefined

FAVOC is ultra-portable. The device is relatively lightweight, weighing only a few pounds and is relatively small compared to other similar devices. The size and weight guarantee that the device can be used anywhere.


Powered by Raspberry Pi

FAVOC uses the Raspberry Pi 3 Model B to control the device. The Raspberry Pi provides power and is directly responsible for controlling the image processing (in association with openCV) that the device uses.
